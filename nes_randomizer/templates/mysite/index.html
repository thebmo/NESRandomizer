Welcome to the NES randomizer. This site was built entirely from scratch using Python's Django 1.6 framework. This site is aimed at users who wish to easily build, maintain, and beat their collection of NES titles. This project originally started out as a spreadsheet, then evolved into a basic python application, and finally a web app. This site is the creation of a labor of love as well as self-educational purposes. (I am self taught)

The NES randomizer itself is a form with built in filters. Though you do not have to be logged in to use the genre filters, you will need to create an account to use the beaten and owned game selection filters. Please note the "Pick Some" feature is not yet enabled.

You can maintain your owned and beaten titles via the Profile link int he top right of the menu once you are are registered. Please send any found bugs or suggestions to NESRandomizer@gmail.com so that I can take a look. Please note that this site is still under heavy development.
